HTML & CSS Basics

  • HTML creates the structure of a webpage.

Common HTML Tags

  • <h1> to <h6> - Headings, from largest (h1) to smallest (h6).
  • <p> - Defines a paragraph.
  • <br> - Inserts a line break.
  • <b> - Makes text bold for emphasis.
  • <i> - Italicizes text.
  • <u> - Underlines text.
  • <a href="URL"> - Creates a hyperlink to another page.
  • <img src="path" alt="description"> - Embeds an image.
  • <ul> and <ol> - Creates lists (bulleted or numbered).
  • <div> - A block-level container.
  • <span> - An inline container for styling text.
